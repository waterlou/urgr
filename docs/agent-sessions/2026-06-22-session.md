### New files

- `libs/rom-scraper/src/sources/mobygames.rs` ‚Äî MobyGames scraper (1 req/s). Searches by name via `/games?title=`, gets details via `/games?id=`. Returns covers, screenshots, genres, moby_score, platforms, alternate_titles.
- `libs/rom-scraper/src/sources/retroachievements.rs` ‚Äî RetroAchievements scraper (4 req/s). `search_by_hash()` uses MD5 across 42 platform‚Üíconsole-ID mappings with disk-cached game lists (`{cache_dir}/retroachievements/{console_id}.json`). `search_by_name()` does local substring match against cached lists. `get_game_detail()` uses `API_GetGameExtended.php`.
- `libs/rom-scraper/src/sources/steamgriddb.rs` ‚Äî SteamGridDB scraper (no rate limit). `search_by_name()` via `/search/autocomplete/`, `get_game_detail()` via `/grids/game/{id}` with Bearer auth. Returns covers/logos only. Filters NSFW/humor/epilepsy tagged grids.

## Design Decisions
- RateLimiter is a shared utility usable by any scraper.
- All scrapers opt-in via env vars (no default API keys).
- RetroAchievements hash index cached to disk JSON per platform (no TTL, easily refreshed by deleting cache dir).
- SteamGridDB is artwork-only; textual fields are empty strings.
- Source priorities: MobyGames=180, RetroAchievements=330, SteamGridDB=380 (sits between existing sources).
- Search by name for RetroAchievements uses local substring match against cached game lists ‚Äî no extra API call.
