## Key Design Decisions
- **Scan never needs import_dir**: Always uses `data/roms/{collection_folder}`
- **CLIs write to `scanned_games`, server reads and updates `game_state`**: Consistent pattern for NPS and DAT
- **Reuse = prior version dirs**: Server checks `{collectionDir}/{priorVersion}/roms/` for each matched game
- **Only older versions considered prior**: `sv.id < current_version_id`
- **Per-ROM availability**: From `download_queue` completed entries, shown as âś“/âś— in ROM table
- **`--game-id` for single-game scan**: Used by downloader after file completes
- **Visibility refresh**: GameDetail re-fetches when tab becomes visible
