# Session: 2025-05-25

## Goal
Add FBNeo (Final Burn Neo) and FB Alpha DAT version support alongside existing MAME support. This includes:
- Reading GitHub tags as FBNeo versions (v1.0.0.0, v1.0.0.02)
- Treating HEAD as "nightly" (refreshes on re-import)
- Hardcoding two old FB Alpha versions (0.2.97.43, 0.2.97.44) common on slower retro consoles

## Files Changed
- `apps/rom-manager-ui/server/index.js` — Major rework of `/api/versions/available` and `POST /api/versions/import-online`:
  - Added `source` query param support to `GET /api/versions/available` (MAME, FBNeo, FBAlpha43, FBAlpha44)
  - FBNeo source fetches tags from `libretro/FBNeo` GitHub API, adds "nightly" for HEAD
  - Added two hardcoded FB Alpha versions from `barbudreadmon/fbalpha-backup-dontuse-ty` and `libretro/fbalpha`
  - Extended `POST /api/versions/import-online` with `source` field to handle FBNeo/FBA download flow
  - FBNeo/FBA import downloads multiple DAT files from GitHub raw content, parses each, deduplicates games
  - FBAlpha43 uses its combined single DAT file; FBNeo and FBAlpha44 use per-system DATs
  - Cache busting on import: FBNeo cache cleared separately from MAME cache
- `apps/rom-manager-ui/src/api.js` — `getAvailableVersions()` and `importOnlineVersion()` accept optional `source` parameter
- `apps/rom-manager-ui/src/components/CollectionDetail.jsx` — DAT versions section now handles FBNeo/FBA alongside MAME; determines source from `collection.folder`; passes source to API calls

## Commands Run
```bash
npm run build             # Build frontend (Vite)
# Test server endpoints:
curl /api/versions/available?source=FBNeo   # Returns nightly + tags + FBA versions
curl /api/versions/available?source=FBAlpha44
POST /api/versions/import-online (source=FBNeo, version=nightly)  # 22,539 games
POST /api/versions/import-online (source=FBAlpha43, version=0.2.97.43)  # 5,245 games
POST /api/versions/import-online (source=FBNeo, version=v1.0.0.0)  # 13,484 games
POST /api/versions/import-online (source=FBAlpha44, version=0.2.97.44)  # 10,640 games
```

## Test Results
- FBNeo nightly: 22,539 games from 17 DAT files in ~7.5s
- FBNeo v1.0.0.0: 13,484 games (tagged release, fewer systems)
- FBAlpha 0.2.97.43: 5,245 games (combined single DAT)
- FBAlpha 0.2.97.44: 10,640 games (per-system DATs, fewer games than FBNeo)
- MAME backward compatible: 286 versions returned, imports unchanged
- Available endpoint correctly tracks imported status across MAME, FBNeo, FBAlpha sources
